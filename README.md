# Strategy Lab

Backtest, simulate, and optimize trading strategies.

## Features
- **Backtester** — Visual rule builder + Pine Script-style code editor
- **Monte Carlo** — 1,000 simulations, probability cone, risk of ruin
- **Exchange Connect** — Import trades via CSV or Hyperliquid API
- **AI Advisor** — BYOK Anthropic API for strategy analysis
- **Trade Journal** — Mood tracking, tags, persistent entries
- **Risk Lab** — Kelly criterion, position sizing, strategy health

## Deploy to Vercel

```bash
npm install
npm run build
```

Or just push to GitHub and connect to Vercel — it auto-detects Vite.

## Data Storage
All data stays on your device (localStorage). Use Export/Import to transfer between devices.

No accounts. No servers. No data collection.
